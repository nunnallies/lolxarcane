## Contexte et Objectifs
Ce projet a été créé dans le but de :
1. Démontrer une compréhension approfondie de Python et SQL.
2. Montrer une capacité à résoudre  gérer des bases de données volumineuses efficacement.
3. Illustrer mon approche de la conception de systèmes, le traitement des données, etc.
4. Voir l'impact de la série Arcane sur le choix des persos en partie. 

### Exemple d'application concrète
> **Application :** Analyse des tendances après un patch 
> Avec les commandes existantes le projet permet de recueillir des données sur les tendances des joueurs en champ select. Avec une mise à jour sur quelques commandes le projet pourra aussi être utilisé pour faire des statistiques sur les joueurs pro lors de compétition connues.   

---

## Fonctionnalités
Ce projet inclut :
- **Récupération automatique des données** depuis une API externe (RIOT).
- **Stockage structuré et optimisé** des données dans une base de données SQL.
- **Analyses de données clés**, telles que :
  - Durée moyenne des parties.
  - Répartition des personnages joués.
- **Export des résultats** en CSV pour une utilisation externe.
